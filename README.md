## Problemas encontrados y mejoras realizadas
Durante la creación de la web, algunos de los problemas encontrados fueron:

- **Problema con la instalación de MkDocs**: Fue necesario usar un entorno virtual para evitar problemas de permisos.
- **Configuración del tema**: Al principio, el tema `material` no se reconocía, pero se solucionó instalando correctamente el paquete.

Mejoras realizadas:
- Agregué una estructura de navegación clara en el archivo `mkdocs.yml`.
- Añadí ejemplos de proyectos, formación y contacto para que la web sea más completa y útil.
